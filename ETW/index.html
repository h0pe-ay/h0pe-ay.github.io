ETW的简介

Event Trace for Windows(ETW)是内核级别的事件追踪机制。

ETW由三部分组成

  • Controllers（事件控制器），用来开关event trace 会话 和 Providers。
  • Providers（事件提供器）， 用来提供事件。
  • Consumers（事件消耗器），用来处理事件。

使用

这里查询Microsoft-Windows-Kernel-Process

  • 第一个表格：GUID以及Provider的信息
  • 第二个表格：Provider的Flag
  • 第三个表格：安全等级

使用session

  • 建立 Session，logman create trace <session name> -p <provider name / GUID> <flags> <security level> -o <path to log file>
  • 启用 Session，logman start <session name>
  • 停用 Session，logman stop <session name>
  • 刪除 Session，logman delete <session name>
